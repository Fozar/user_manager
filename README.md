# server
```
cd server
```

### Create venv
##### Windows:
```
python -m venv venv
cd venv/Scripts
activate.bat
cd ../../
```
##### Linux:
```
python3 -m venv venv
source venv/bin/activate
```

### Install requirements
##### Windows:
```
pip install -r requirements.txt
```
##### Linux:
```
pip3 install -r requirements.txt
```

### Customize configuration in config.py!

### Run server
##### Windows:
```
python app.py
```
##### Linux:
```
python3 app.py
```
